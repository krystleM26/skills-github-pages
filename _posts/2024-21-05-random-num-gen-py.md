```python
import random

Function Definitions:
random.randit(min, max) - returns a random integer between min and max, inclusive
random.random() - return a random real number (float) between 0 and 1
random.uniform(min, max) Returns a random real number (float) between min and max
random.seek(x)- sets the seed of random number generator to x.
